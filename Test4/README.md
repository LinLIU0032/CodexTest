# Test4 - Random Joke Generator

## 项目简介
一个纯前端随机笑话生成器，点击按钮后从预设笑话列表中随机展示中英文内容。

## 功能
- 随机显示笑话（中英双语）
- 避免连续两次展示同一条笑话
- 简洁卡片式页面布局

## 技术栈
- HTML
- CSS
- Vanilla JavaScript

## 文件结构
```text
Test4/
  index.html
  style.css
  script.js
  README.md
```

## 运行方式
直接用浏览器打开 `index.html`。
